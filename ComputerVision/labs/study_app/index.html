Midterm-Style Sample Question from-scratch NumPy answer + output

Apply edge detection math followed by thresholding to generate a binary image with pixels as 0s and 1s. Do not use OpenCV built-ins.

import numpy as np  # Bring in NumPy so we can do image math.

patch = np.array([  # Make a 3x3 image patch with a bright right edge.
    [0, 0, 255],  # Top row.
    [0, 0, 255],  # Middle row.
    [0, 0, 255]  # Bottom row.
], dtype=np.float32)  # Use floats so multiplication works cleanly.
sobel_x = np.array([[-1, 0, 1], [-2, 0, 2], [-1, 0, 1]])  # Kernel that detects vertical edges.

gx = np.sum(patch * sobel_x)  # Multiply patch by kernel and add the result.
binary = int(abs(gx) > 500)  # Strong edge becomes 1; weak edge becomes 0.

print(int(gx))  # Print the edge strength.
print(binary)  # Print the final binary pixel.
